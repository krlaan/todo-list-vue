# todo-list-vue
Frontend for Todo List using Vue.js,vimplementing CRUD operations. Integrated with: https://taltech.akaver.com/swagger/index.html (JavaScript course)


## Functionality:
- Implemented simple task/todo management system
- Implemented inital auth flow (register and login)
- Implemented three crud views against rest endpoint (must require auth)
- Implemented filtering based on completed and archieved status
- Provided hidden prohibited word-list admin UI


## Running the project:

```sh
npm run dev
```
